# EVED
*PHP web project for records, archives, and everyday tools.*

**EVED** is a PHP web project that brings together several practical tools for
keeping structured records and managing private archives. It includes an address
directory for people, organizations, contacts, postal addresses, groups, and
notes, together with a film scan gallery for presenting and administering
digitized negatives and slides.

The project is designed for a current PHP runtime and a configured web server
and database. Its focus is on keeping useful data and workflows directly under
the control of the application instead of depending on external hosted services.

This repository is best treated as the source code of a working application.
Local data, uploaded media, and deployment details are environment-specific and
are kept outside the repository.

**EVED** is released under the **MIT license**. Included phone and postal address
metadata carry their own third-party license notices in **license.txt**.
